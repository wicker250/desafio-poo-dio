# Desafio DIO – POO com Java

Projeto desenvolvido para praticar os fundamentos de **Programação Orientada a Objetos (POO)** em Java.

## Conceitos aplicados
- Abstração  
- Encapsulamento  
- Herança  
- Polimorfismo  

## Domínio
Modelagem de um **Bootcamp**, com cursos, mentorias e desenvolvedores que evoluem e acumulam XP.

## Tecnologias
- Java 11  
- Git & GitHub
